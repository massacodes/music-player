# Music Player
A music player built with HTML, JavaScript, and TailwindCSS allowing users to play music.

## Features
- Progress Bar
- Navigation Buttons
- Playback Time
- Volume Control

## Tech Stack
- HTML
- JavaScript
- TailwindCSS

## Setup

### Prerequisites
- Node.js
- npm

### Steps
```bash
git clone https://github.com/yourname/music-player.git
cd music-player
npm install
```
